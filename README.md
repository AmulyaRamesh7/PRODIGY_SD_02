# PRODIGY_SD_02
Number Guessing Game – Streamlit App
📌 Project Overview

This is a simple and interactive Number Guessing Game built using Python and Streamlit.

The application generates a random number between 1 and 100, and the user has to guess the correct number. The app provides feedback if the guess is too high or too low and tracks the number of attempts taken to win.

🚀 Features

🎲 Random number generation (1–100)

⬆️ Feedback for high guesses

⬇️ Feedback for low guesses

🔢 Attempt counter

🔄 Restart game button

🌐 Deployable as a web application using Streamlit

🛠️ Tech Stack

Python

Streamlit

Random module
